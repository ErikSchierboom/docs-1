# Line is too long
[!INCLUDE[vs2017banner](../../../visual-basic/includes/vs2017banner.md)]

Source text lines cannot exceed 65535 characters.  
  
 **Error ID:** BC30494  
  
### To correct this error  
  
-   Shorten the length of the line to 65535 characters or fewer.
