# Compiler Error CS1022
Type or namespace definition, or end-of-file expected  
  
 A source-code file does not have a matching set of braces.  
  
 The following sample generates CS1022:  
  
```  
// CS1022.cs  
namespace x  
{  
}  
}   // CS1022  
```
