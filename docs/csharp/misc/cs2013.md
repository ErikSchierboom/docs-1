# Compiler Error CS2013
Invalid image base number 'value'  
  
 An invalid value (not a number) was passed to the [/baseaddress](../../csharp/language-reference/compiler-options/baseaddress-csharp-compiler-options.md) compiler option.  
  
 The following sample generates CS2013:  
  
```  
// CS2013.cs  
// compile with: /target:library /baseaddress:x  
// CS2013 expected  
class MyClass  
{  
}  
```
