# Compiler Error CS1906
Invalid option 'option'; Resource visibility must be either 'public' or 'private'  
  
 This error indicates an invalid [/resource (Embed Resource File to Output)](../../csharp/language-reference/compiler-options/resource-csharp-compiler-options.md) or [/linkresource (Link to .NET Framework Resource)](../../csharp/language-reference/compiler-options/linkresource-csharp-compiler-options.md) command line option. Check the syntax of the **/resource** or **/linkresource** command line option, and make sure that the accessibility modifier used is either **public** or `private`.
