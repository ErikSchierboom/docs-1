# Compiler Error CS0006
[!INCLUDE[csharpbanner](../../../csharp/includes/csharpbanner.md)]

Metadata file 'dll_name' could not be found  
  
 The program was compiled and explicitly passed the name of a file that contained metadata; however, the .dll was not found. For more information, see [/reference (C# Compiler Options)](../../../csharp/language-reference/compiler-options/reference-csharp-compiler-options.md).
